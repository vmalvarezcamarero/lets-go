# Análisis del mercado del diagnóstico in vitro (IVD) en España

Este repositorio contiene el código desarrollado en el marco del Trabajo Fin de Máster titulado **“Análisis del mercado de Diagnóstico In Vitro (IVD) en España: Evolución, distribución territorial y competencia en la contratación pública (2020-2025)”**.

El objetivo de este trabajo es analizar el mercado español del diagnóstico in vitro a partir de datos de contratación pública, con especial atención a la adquisición de equipamiento y consumibles IVD por parte de instituciones sanitarias públicas. El análisis busca caracterizar los patrones de gasto, la distribución territorial, los modelos de contratación predominantes y los principales retos estructurales asociados a la renovación tecnológica, la competencia y la dependencia de proveedor.

El estudio se basa en datos procedentes de plataformas públicas de contratación y de fuentes institucionales a nivel nacional y europeo. Para identificar de forma homogénea las adquisiciones relacionadas con el diagnóstico in vitro, se utiliza un conjunto acotado de códigos CPV (Common Procurement Vocabulary). El repositorio incluye los scripts necesarios para la limpieza, el procesamiento y el análisis de los datos, así como para la generación de las figuras incluidas en el Trabajo Fin de Máster.

Este repositorio tiene como finalidad garantizar la transparencia y la reproducibilidad del análisis realizado. Todas las figuras y resultados presentados en el trabajo se generan directamente a partir del código aquí contenido. La estructura del repositorio separa el procesamiento de datos, los scripts de análisis, los cuadernos de trabajo y los resultados, con el fin de facilitar su comprensión y reutilización.

Cabe señalar que los archivos de datos originales no se incluyen cuando están sujetos a limitaciones de tamaño o de licencia; no obstante, todas las fuentes de datos y los pasos de procesamiento se documentan de forma detallada para permitir la replicación del estudio.



# Analysis of the In Vitro Diagnostics (IVD) Market in Spain

This repository contains the code developed as part of the Master’s Thesis entitled **“Análisis del mercado de Diagnóstico In Vitro (IVD) en España: Evolución, distribución territorial y competencia en la contratación pública (2020-2025)”**.

The objective of this work is to analyse the Spanish IVD market through publicly available procurement data, with a particular focus on the acquisition of in vitro diagnostic equipment and consumables by public healthcare institutions. The analysis aims to characterise expenditure patterns, territorial distribution, procurement models, and structural challenges related to technological renewal, competition, and supplier dependency.

The study is based on data obtained from public procurement platforms and institutional sources at both national and European levels. A restricted set of CPV (Common Procurement Vocabulary) codes is used to identify IVD-related purchases in a homogeneous and reproducible manner. The repository provides the scripts required to clean, process, and analyse the data, as well as to generate the figures included in the Master’s Thesis.
